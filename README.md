# Hangarin
**Task & To-Do Manager built with Django**

Hangarin is a modern task management web application that helps users organize tasks, subtasks, notes, categories, and priorities through a clean dashboard interface.

---

## Features

- Task management (Create, Edit, Delete)
- Subtasks support
- Notes system
- Categories and priorities
- Search and filtering
- Dashboard statistics
- Dark/Light mode UI
- User authentication
- Google OAuth login
- GitHub OAuth login

---

## Tech Stack

- Python
- Django
- HTML / CSS / JavaScript
- SQLite
- django-allauth (OAuth authentication)
